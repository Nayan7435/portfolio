# Portfolio Website

Personal portfolio website showcasing my projects, skills, and internship experience as a final-year B.Tech Computer Engineering student.

🔗 **Live demo:** [nayan7435.github.io/portfolio](https://nayan7435.github.io/portfolio) <!-- update once deployed -->

## Features

- Dark, developer-themed UI with neon accent highlights
- Fully responsive layout (mobile, tablet, desktop)
- Smooth scrolling navigation with active section highlighting
- Typing animation for hero subtitle
- Sections for About, Skills, Projects, Internships, Resume, and Contact
- One-click resume download
- Contact form that opens a prefilled email draft (no backend required)

## Tech stack

- **HTML5** — semantic structure
- **CSS3** — custom properties, Flexbox, Grid, responsive media queries
- **JavaScript (vanilla)** — DOM manipulation, scroll events, form handling

## Run locally

```bash
git clone https://github.com/Nayan7435/portfolio.git
cd portfolio
```

Open `index.html` in your browser, or serve it with a live server extension for hot reload.
